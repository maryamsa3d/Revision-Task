# Revision-Task

## Project Description
This project analyzes a library dataset using Python, Pandas, and NumPy. The goal is to understand borrowing patterns and generate useful insights.

## Dataset
The project uses three CSV files:
- books.csv
- members.csv
- borrowings.csv

## Libraries Used
- Pandas
- NumPy
- Matplotlib

## How to Run
1. Install the required libraries.
2. Open Revision_Task.ipynb.
3. Run all cells from top to bottom.
